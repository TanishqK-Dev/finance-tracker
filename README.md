🚀 CashFlow: Personal Finance Tracker
Tagline: A lightweight, privacy-focused expense manager built for the modern web.



📝 Project Overview

CashFlow is a functional web application designed to help users monitor their daily income and expenses. It features a clean, mobile-first interface and provides real-time financial calculations to keep users on top of their budget.



✨ Key Features

Real-Time Math Engine: Automatically calculates total balance, income, and expenses using JavaScript's .reduce() and .filter() methods.

Persistent Data: Uses localStorage to save transaction history, ensuring data remains available even after closing the browser or refreshing the page.

Categorization: Allows users to tag transactions (e.g., Food, Salary, Rent) for better organization.

Responsive UI: Fully optimized for both desktop and mobile devices with a sleek, dark-minimalist aesthetic.

Safety Checks: Includes confirmation dialogs before deleting data to prevent accidental loss.



🛠️ Tech Stack

Frontend: HTML5, CSS3 (Flexbox/Grid)

Logic: Vanilla JavaScript (ES6+)

Storage: Browser LocalStorage API
